# 🚀 SpaceX Falcon 9 Landing Prediction

## 📌 Project Overview
This project analyzes SpaceX Falcon 9 launch data to predict the success of first-stage landings using data science and machine learning techniques.

---

## 🎯 Objectives
- Analyze launch data using EDA and SQL
- Visualize launch trends and success patterns
- Build interactive dashboards using Plotly Dash
- Predict landing success using classification models

---

## 📊 Data Collection
- SpaceX REST API
- Web scraping (Wikipedia)
- Data processed and stored using Pandas

---

## 🔍 Exploratory Data Analysis (EDA)
- Launch success trends over time
- Payload vs success relationship
- Orbit type and launch site analysis

---

## 🗺️ Interactive Visualizations
- Folium maps for launch site analysis
- Plotly Dash dashboard for interactive insights

---

## 🤖 Machine Learning Models
- Logistic Regression
- Support Vector Machine (SVM)
- Decision Tree
- K-Nearest Neighbors (KNN)

✔️ All models achieved ~83% accuracy

---

## 📈 Key Insights
- Launch success improved over time
- Payload and orbit type impact landing success
- Certain launch sites show higher reliability
